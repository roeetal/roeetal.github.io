Eduhacks

Language is difficult to learn because it is fluid and has far less structure then many other subjects. Many children have difficulty recognizing the patterns inherent in languages, and so learning a language becomes an arduous task for them. The theme for the hackathon was education and we decided to develop a comprehension tool designed to help children with this difficulty.
Using Google's Syntaxnet (a Tensorflow NLP library) we created a web application that was able to parse, "comprehend", and manipulate a text. Using linguistics algorithms, we developed a question generator to content based questions subtly. If the reader answers the questions correctly, they could simply continue. If they answered incorrectly, the application would help the user by providing hints via the text by highlighting or underlining words. These hints target repetitive conceptual flaws that the reader makes. The app attempts to force the reader to recognize certain patterns in the language that were otherwise missed.
