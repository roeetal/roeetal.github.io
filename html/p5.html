Eduhacks

Language is difficult to learn because it is fluid and has far less structure then many of the other subjects we learn. Many children have difficulty recognizing the patterns inherent in languages, and so learning a language becomes an arduous task for them. The theme for the hackathon was education and we decided to develop a comprehension tool designed to help children with the aforementioned problem.
Using Google's Syntaxnet (a Tensorflow library for natural language processing) we created a web application that was able to parse and manipulate a text. And using linguistics algorithms, we developed a question generator that can generate relevant questions from the content in the text. The motivation behind the web app was to ask the reader a question after very few paragraphs. If the reader answered correctly, they would simply continue. If they answered incorrectly, the application would use interact with the user via the text by highlighting or underlining words as hints. The app would also try recognize if there are repetitive conceptual flaws in the reader's answer and target the relevant content for its hints. The app attempts to force the reader to recognize certain patterns in the language until the reader begins to understand them.
